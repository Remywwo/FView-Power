FEATURES

为思考而设计

没有花哨的功能，只有让你专注于内容的工具。

01

Markdown 滚动同步

编辑器与预览实时联动，目录自动浮现在右侧，悬停即可跳转。

02

HTML 预览能力

本地 iframe 沙箱渲染，CSS / JS 完整保留，零网络请求。

03

文件夹浏览

左侧树形目录，自动跳过 node_modules、target 等构建产物。拖拽文件夹即可打开。

04

PDF 大纲导航

支持 ⌘G 跳页、方向键翻页、大纲浮动面板实时高亮当前位置。

05

拖拽打开

把文件或文件夹直接拖进窗口，识别文件类型、立即渲染。

06

可定制排版

字体、字号、行高、主题（亮 / 暗）自由组合，配置本地保存。

SCREENSHOTS

界面即内容

没有 chrome、没有装饰，每一像素都为内容服务。

SHORTCUTS

键盘即速度

离开鼠标，让一切都在指尖完成。

FAQ

常见问题

FView Power 是免费的吗？

是的，完全免费且开源。代码以 MIT 协议发布在 GitHub。

支持哪些文件类型？

Markdown (.md)、PDF、纯文本 (.txt)、代码文件（.js/.ts/.py/.go/.rs 等 30+ 种）、HTML、常见图片格式。

需要联网吗？

不需要。所有渲染都在本地完成，文件不会上传到任何服务器。

如何贡献？

欢迎在 GitHub 提 issue 或 PR。代码改动请遵循现有 TypeScript / React 编码风格。
