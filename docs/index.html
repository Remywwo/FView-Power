FEATURES

为思考而设计

没有花哨的功能，只有让你专注于内容的工具。

01

多格式编辑与预览

Markdown、20+ 种代码语言、HTML 均可编辑；PDF、9 种图片格式流畅预览。一个窗口覆盖日常工作全场景。

02

AI 助手

上下文感知对话：Markdown 全文自动注入，PDF 目录与页面文本实时提取。支持总结、翻译、代码解释。兼容 OpenAI / Anthropic / Ollama 等任意接口。

03

Markdown 工作台

分屏 / 编辑 / 预览三种视图，滚动同步，浮动目录自动高亮。GFM 表格与任务列表、KaTeX 数学公式、Mermaid 图表、代码高亮、emoji 短码——开箱即用。

04

PDF 阅读增强

大纲浮动面板实时定位当前章节，⌘G 快速跳页，方向键翻页，缩放自如。AI 自动提取目录结构和页面文本作为对话上下文。

05

文件夹浏览

左侧树形目录，拖拽文件夹即可打开。自动跳过 node_modules、.git、target、dist 等构建产物，快速定位目标文件。

06

由你做主

亮 / 暗主题，9 款字体，8–72px 字号，可调行高，English / 中文界面。AI 提供商与模型自由配置。所有设置本地持久化。

SCREENSHOTS

界面即内容

没有 chrome、没有装饰，每一像素都为内容服务。

SHORTCUTS

键盘即速度

离开鼠标，让一切都在指尖完成。

FAQ

常见问题

FView Power 是免费的吗？

是的，完全免费且开源。代码以 MIT 协议发布在 GitHub。

支持哪些文件类型？

Markdown (.md)、PDF、纯文本 (.txt)、代码文件（.js/.ts/.py/.go/.rs 等 30+ 种）、HTML、常见图片格式。

需要联网吗？

不需要。所有渲染都在本地完成，文件不会上传到任何服务器。

如何贡献？

欢迎在 GitHub 提 issue 或 PR。代码改动请遵循现有 TypeScript / React 编码风格。
